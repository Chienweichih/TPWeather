熱門景點的臉書打卡數與天氣指數的關係之研究
-----------------------------------

本研究要探討地區性人氣活躍程度，是否和社群網站的打卡數量以及天氣的因素有所關連。

我們利用 Facebook 的 API 取得景點的打卡數量、利用 Python 撰寫爬蟲程式抓取台北市國小的氣象資料。

以迴歸分析計算出的數學模型觀察這些資料之間的關聯，進而估測出特定景點的 Point of Interest (POI) 值。
